# 🛡️ DevSecOps Security Monitoring System

A Dockerized DevSecOps project that simulates cyber attacks, detects suspicious activity using a Flask application, visualizes security metrics through Prometheus and Grafana dashboards, and integrates GitHub Actions-based CI/CD automation for testing and security scanning.

## Project Workflow

Fake Attack Simulation
↓
Flask Security Monitoring App
↓
Prometheus Metrics Collection
↓
Grafana Security Dashboard

────────────────────────────

Git Push
↓
GitHub Actions CI/CD
↓
pytest Automated Tests
↓
Bandit Security Scan

## Features

* Brute force login detection
* SQL injection detection
* Suspicious path detection
* Security headers implementation
* Real-time metrics monitoring
* Grafana dashboard visualization
* Dockerized multi-container architecture
* GitHub Actions CI/CD integration
* Automated testing with pytest
* Python security scanning using Bandit

## Technologies Used

Python, Flask, Docker, Docker Compose, Prometheus, Grafana, Git, GitHub, GitHub Actions, pytest, Bandit, YAML
